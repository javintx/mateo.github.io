# 🏀 La página de Mateo

Página web personal de Mateo, hecha para aprender a programar.

## 🌐 Ver la página

👉 <a href="https://javintx.github.io/mateo.github.io/" target="_blank" rel="noopener noreferrer">Abrir página web</a>

## 📁 Archivos

| Archivo | Para qué sirve |
|---|---|
| `index.html` | El contenido de la página (textos, emojis, secciones) |
| `estilo.css` | El aspecto visual (colores, tamaños, bordes) |

## ✏️ Cómo editar la página

1. Abre `index.html` con el **bloc de notas** para cambiar los textos
2. Abre `estilo.css` con el **bloc de notas** para cambiar los colores
3. Abre `index.html` con el **navegador** para ver el resultado
